## 需求
1. 前端项目docker部署 CI/CD
2. 快速搭建docker开发环境
3. docker node微服务开发部署 

## 心得
docker 是一个CS模式的服务
1. 有客户端和服务端
2. 客户端和服务端可以不在同一台主机上，但要求版本一直，可通过远程ssh连接，本地通过linux内部通信连接
